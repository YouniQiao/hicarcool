# 检查手机

您可以检查一下您的手机是否支持Next HiCar，或者重新购买一部支持Next HiCar的手机。

## 1.哪些手机品牌支持Next HiCar

由于Next HiCar是华为提供的互联方案，所以当前只有华为运行HarmonyOS 5及以上系统的手机支持Next HiCar。

## 2.哪些手机型号支持Next HiCar

具体支持Next HiCar的手机型号，可以在官网查看，请查看[华为开发者联盟-HiCar支持的手机](https://developer.huawei.com/consumer/cn/doc/development/HiCar-Guides/available-phones-0000001227954439)


## 3.查看您的手机是否支持Next HiCar

查看华为手机是否支持Next HiCar的最佳的方式如下：

打开手机“设置 > 多设备协同”，如果能够看到“HUAWEI HiCar”菜单，就表示您这部手机是支持Next HiCar的。

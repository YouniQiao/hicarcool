# 切换音频播放设备

Next HiCar在连接成功后，默认会将音频切换到车载音响设备进行播放。您可以根据需要将音频播放设备在手机（包括与手机连接的耳机等）和车载音频设备间进行切换。

## 手机通知栏快速切换

1. 在Next HiCar连接成功后，点击实况窗。

2. 在弹出的Next HiCar横幅中，点击右侧第二个按钮，切换音频播放设备。
   
    <img
        src={require('./img/switch-audio-device-1.jpg').default}
        width="400" 
    />
   
    切换到车机后，再次点击后将把音频切换到手机播放。

    切换到手机后，如果手机同时连接耳机、智能眼镜等设备，音频会自动从耳机、智能眼镜等设备播放。

   
    

## 通话中快速切换

Next HiCar连接后，您可以在通话过程中将通话切换回手机或与手机连接的其它设备，如耳机、智能眼镜等。

1. 您可以在通话界面，从左向右，选择第三个按钮，进入设备选择列表。
   
    <img
        src={require('./img/switch-audio-device-2.jpg').default}
        width="800" 
    />

2. 在设备选择列表，根据需要将通话切换到相应的输出设备。

        src={require('./img/switch-audio-device-3.jpg').default}
        width="800" 
    />

# 摇一摇应用流转

Next HiCar支持通过摇一摇手机，将手机中的应用流转到车机上。

## 设置方式

您可以通过如下任一方式开启摇一摇导航流转：

* 进入手机“设置 > 多设备协同 > HUAWEI HiCar”，点击我的设备中已连接的设备，打开摇一摇应用流转开关。
  
    <img
        src={require('./img/shake-app-1.jpg').default}
        width="400" 
    />

* 在HiCar连接后，在车机侧进入“设置 > 摇一摇导航流转”，打开摇一摇应用流转开关。


## 操作步骤

1. 在手机侧打开支持摇一摇应用流转的应用，手机实况窗即可显示“可流转”字样。
   
    <img
        src={require('./img/shake-app-2.jpg').default}
        width="400" 
    />

    在横幅中，也会显示一个“流转”按钮。

    <img
        src={require('./img/shake-app-3.jpg').default}
        width="400" 
    />

2. 摇一摇手机，Next HiCar车机侧将会自动打开对应的应用。

# 触控板



当您在连接Next HiCar的状态下，打开手机上的驾驶模式功能，您将看到驾驶模式界面变成如下HiCar遥控器的样式，。

该功能尤其适合车机侧非触摸屏的Next HiCar用户。

## 进入触控板

在Next HiCar连接状态下，选择已连接的设备后，点击“触控板”即可。

## 触控板功能

在触控板中可以实现：

* 直达导航：底部按钮栏中第一个按钮，点击后可以直接进入地图应用。
* 直达音乐：底部按钮栏中第三个按钮，点击后可以直接进入音频应用。
* 后退：底部按钮栏中第二个按钮，点击后可以后退一步。
* 唤起语音：右上角的麦克风图标，可以唤起智慧语音。
* 触控操作：在空白区域滑动+点击，即可在车机界面上完成相应的操作。

# 桌面

桌面是您日常进入Next HiCar时显示的第一个界面。

## 智慧桌面

智慧桌面布局中，地图卡片更突出，可以直接在卡片中显示更详细的地图信息，支持在其他卡片区域上下滑动查看更多卡片。

<img
    src={require('./img/desktop-1.jpg').default}
    width="800" 
/>

您可以根据喜好设置地图卡片的左右显示位置。

<img
    src={require('./img/desktop-2.jpg').default}
    width="800" 
/>


## 切换桌面布局

您可以在“设置 > 桌面布局”中选择您偏好的布局方式，应用后立即生效。

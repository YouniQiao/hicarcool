# 熄屏显示

Next HiCar支持更换熄屏显示，直接在车机界面中简约显示日期和时间。


## 进入/退出熄屏显示

您可以在Next HiCar连接后，点击DOCK区中“熄屏显示”图标进入熄屏显示状态，再次点击后退出熄屏显示状态。

## 熄屏显示壁纸更换

Next HiCar支持更换桌面壁纸，您可以在Next HiCar连接后，点击DOCK区上方第一个图标切换到应用桌面，进入 “主题”，切换到“熄屏显示”页签，您可以选择熄屏显示时的界面效果。

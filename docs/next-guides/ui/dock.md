# DOCK区

<img
    src={require('./img/desktop-1.jpg').default}
    width="800" 
/>


Next HiCar在DOCK区提供了系统信息展示区域及核心功能的快捷按键，包括：
* 时间/信号/电量：显示相应内容，不支持点击操作。
* 应用/首页：点击可在应用图标页和首页之间切换。
* 导航：点击切换到当前地图应用，再次点击弹出地图应用列表，可快速切换不同地图应用。
* 音频：点击切换到当前音频应用，再次点击弹出音频应用列表，可快速切换不同音频应用。
* 最近使用：显示最近使用的应用。
* 熄屏显示：点击可以切换到全屏熄屏显示状态。
* 语音球：点击可以唤醒智慧语音，进行语音操控。


:::tip

1. 播放长视频时，DOCK区会自动隐藏。
2. 您可以在DOCK区上下滑动，快速切换应用图标。

:::

# 桌面卡片

桌面卡片是Next HiCar提供的一种快捷操作方式，您可以通过卡片快速了解应用/服务的相关信息、进入应用/服务以及执行应用/服务提供的快捷功能。

## 卡片分类

<img
    src={require('./img/desktop-1.jpg').default}
    width="800" 
/>

卡片主要包括以下几类：
* 地图卡片：显示地图应用快捷功能，如回家、去公司等，以及在导航中显示导航简要信息。
* 音频卡片：显示音频应用快捷功能，包括上一首、播放/暂停、下一首。
* 日历卡片：显示日历应用中当天的日程安排信息。
* 天气卡片：显示天气应用中当前定位位置的天气信息。
* 智能推荐卡片：显示应用推荐、技巧推荐等信息。


:::tip

卡片由系统自动生成及排序，无需人工干预。这也意味着您无法手工调节卡片的显示顺序。
:::


## 卡片编辑

长按桌面卡片后，可以进入卡片编辑页面。通过点击卡片右上角的添加和删除按钮，可以快速编辑卡片。

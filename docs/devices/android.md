# 原车Android系统可以安装软件

## 适用场景

适用于原车不支持HiCar，也不支持CarPlay和CarLife，但车机系统是Android系统，并可以自行安装应用的车型。您可以通过自行在车上安装软件，然后配合盒子使用HiCar。

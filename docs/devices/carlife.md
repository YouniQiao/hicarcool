# 原车支持有线CarLife

## 适用场景

适用于原车不支持HiCar，但支持百度CarLife的车型，因为现在有第三方的转换盒，可以通过CarLife通道，让您原本不支持HiCar的车型也可以使用上HiCar。

## 检查方法

您在上车启动车辆后，在您车机的系统中查找CarLife相关的菜单或按钮，一般在互联或者多媒体相关的子菜单里面。

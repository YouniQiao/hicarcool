# 功能差异

为了有效区分，我们将运行在HarmonyOS 4.x以下系统上的HiCar称为经典HiCar，运行在HarmonyOS 5及以上系统（即纯血鸿蒙）上的HiCar称为Next HiCar。

Next HiCar与经典HiCar存在一些功能差异，所以本章节内容会根据Next HiCar的功能演进情况持续刷新。

:::tip

功能差异确认时间点：2025.11.16

::: 

## 连接

| 功能点          | 经典HiCar         | Next HiCar                    |
| ----------     | ---------- | ------------------------- | 
| 无线连接        | 支持 | 支持               | 
| 有线连接        | 支持 | 支持，系统需升级到5.0.0.126 SP8及以上版本   | 
| 自动连接        | 支持 | 支持 | 



## 界面

| 功能点          | 经典HiCar         | Next HiCar                      |
| ----------- | ---------- | ------------------------- | 
| 桌面壁纸        | 支持 | 支持               | 
| 熄屏显示       | 支持，14.2.0.350及以上版本支持 | 支持   | 
| 返回车机界面        | 支持 | 支持 | 



## 应用

| 功能点          | 经典HiCar         | Next HiCar                      |
| ----------- | ---------- | ------------------------- | 
| 调整应用显示顺序        | 支持 | 支持               | 

## 语音

| 功能点          | 经典HiCar         | Next HiCar                      |
| ----------- | ---------- | ------------------------- | 
| 语音功能        | 支持 | 支持               | 


## 功能

| 功能点          | 经典HiCar         | Next HiCar                      |
| ----------- | ---------- | ------------------------- | 
| 设置家和工作地址        | 支持 | 支持               | 
| 摇一摇导航流转        | 支持 | 支持   | 
| 驾驶模式        | 支持 | 不支持 | 
| 截屏        | 支持 | 不支持 |

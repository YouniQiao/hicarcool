# 手机遥控器  

HiCar NEXT支持在手机上通过遥控器操控车机侧的HiCar界面。对于非触屏的车机来说，这个功能将会很有用。

当我们连接HiCar后，在手机侧的设备管理中可以看到“遥控”功能入口。

进入遥控菜单之后，支持的功能如下：

* 操控面板：占据了界面最大位置的操控面板，可以滑动和点击，模拟在车机界面上进行操作。
* 导航按钮：点击后进入导航应用。
* 返回箭头：点击后返回上一步操作。
* 音乐按钮：点击后进入音乐应用。
* 小艺按钮：点击后出发小艺语音，再次点击取消。

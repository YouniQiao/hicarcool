# 调整应用顺序

您可以根据您的喜好，自由调整连接HiCar后，应用图标页中应用的显示顺序。

1. 根据HiCar是否连接，选择不同的方式进入到应用管理界面。
    * 已连接HiCar时，从手机屏幕顶部下滑出通知中心，在通知栏中点击HiCar智行横幅通知。
    * 未连接HiCar时，进入手机“设置 > 更多连接 > HiCar 智行”，点击“我的设备”下连接过HiCar的设备。

2. 点击“HiCar 智行 应用管理”。

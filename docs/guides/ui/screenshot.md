# 截屏

HiCar截屏功能依赖手机上的驾驶模式功能。如果您目前不了解HiCar驾驶模式，您可以点击[驾驶模式简介](../../dmode/driving-mode.md)了解。

当您在连接HiCar的状态下，打开手机上的驾驶模式功能，您将看到驾驶模式界面变成如下HiCar遥控器的样式，此时，您点击中间的“截屏”按钮即可完成对HiCar界面截屏。截取的图片保存在手机相册中。

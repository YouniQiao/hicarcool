HiCar Cool is the domain name of this site, and it's also the original idea behind building this site — HiCar is indeed pretty Cool.

As currently envisioned, HiCar Cool will mainly focus on several aspects of HiCar:

1. **HiCar APK Downloads.** Every time I see someone asking for the HiCar installation package in a group chat, I think, I need to set up a website dedicated to hosting HiCar installation packages. Of course, if that's not appropriate, I'll remove the download links at any time.

2. **HiCar Tutorials.** As one of the earliest adopters of HiCar, from the end of 2019 until now, it's been quite a while. I consider myself a seasoned HiCar experience officer. I want to write down some methods, tips, and precautions for using HiCar, and share them so that more and more people can use HiCar and enjoy the fun it brings.

3. **Latest HiCar News.** I'll gather HiCar-related news, including but not limited to the latest HiCar version updates, new HiCar partnership requirements, etc.

4. **HiCar Ecosystem.** As the HiCar ecosystem gradually expands, there are more and more HiCar-compatible devices. I'll try my best to collect information about HiCar-related devices and accessories. As time and budget allow, I'll purchase some HiCar peripherals and create review content. This might be slower since both time and money are limited resources.

That's roughly the plan for the site content, though of course it may change over time.

## Why Build This Site?

Probably out of passion. As a hardcore HiCar enthusiast, I've invested a huge portion of my spare time into HiCar — running a WeChat public account, writing on Zhihu, publishing on Toutiao (Today's Headlines), setting up group chats, and now building this site.

As for why I built this site — maybe because all those channels mentioned above are too fragmented. Deep down, I wanted a complete site to present everything I understand about HiCar.

And so, HiCar Cool was born. That's it.

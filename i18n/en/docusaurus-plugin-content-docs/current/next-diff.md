# Feature Differences

To distinguish between the two, we refer to HiCar running on systems below HarmonyOS 4.x as Classic HiCar, and HiCar running on HarmonyOS 5 and above (i.e., Pure HarmonyOS) as Next HiCar.

There are some feature differences between Next HiCar and Classic HiCar, so this chapter will be continuously updated based on the feature evolution of Next HiCar.

:::tip

Feature differences confirmed as of: 2025.11.16

::: 

## Connectivity

| Feature | Classic HiCar | Next HiCar |
| ---------- | ---------- | ------------------------- | 
| Wireless Connection | Supported | Supported               | 
| Wired Connection | Supported | Supported, system must be updated to 5.0.0.126 SP8 or above |
| Auto Connect | Supported | Supported |

## Interface

| Feature | Classic HiCar | Next HiCar |
| ----------- | ---------- | ------------------------- | 
| Desktop Wallpaper | Supported | Supported               | 
| Always-On Display | Supported, version 14.2.0.350 and above | Supported   | 
| Return to Vehicle Interface | Supported | Supported |

## Apps

| Feature | Classic HiCar | Next HiCar |
| ----------- | ---------- | ------------------------- | 
| Adjust App Display Order | Supported | Supported               |

## Voice

| Feature | Classic HiCar | Next HiCar |
| ----------- | ---------- | ------------------------- | 
| Voice Function | Supported | Supported               |

## Features

| Feature | Classic HiCar | Next HiCar |
| ----------- | ---------- | ------------------------- | 
| Set Home and Work Addresses | Supported | Supported               | 
| Shake to Transfer Navigation | Supported | Supported   | 
| Driving Mode | Supported | Not Supported | 
| Screenshot | Supported | Not Supported |

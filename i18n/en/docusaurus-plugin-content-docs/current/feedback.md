# Site Update Notes

Records of updates to this site's content and functionality, for your quick reference.

Wallpaper now supports custom wallpapers, and the dropdown banner style after refreshing the HiCar connection has been updated.

1. Refreshed the supported phone list, see [Check Phone](/check/phone.md).
2. Added voice screenshot instructions, see [Screenshot](/guides/ui/screenshot.md).

## 2023-06-16

Added the "Development Guide" section, listing official guide links for quick reference, see [Development Guide](/dev).

Removed the "Latest News" section. Future HiCar-related news will continue to be provided via the official WeChat account "HiCar情报站" (HiCar News Station).

## 2023-05-28

1. Added explanation about HiCar vehicle-side versions, clarifying why installing the HiCar vehicle-side app cannot make a non-HiCar-compatible vehicle work with HiCar, see [HiCar Vehicle Version](vehicle.md).
2. Added a HiCar simulator section, attaching the original instructions for using HiCar on a tablet, see [Using HiCar on a Tablet](simulator/box-hicar-pad.md).

Added download links for versions 13.2.0.451 and 13.2.0.455, see [Homepage](https://hicar.cool/). Version 451 is the official version, and 455 is the beta version following the official version, which enables Petal Maps and WeChat on top of 451.

Added HiCar Driving Mode documentation, see [Driving Mode Introduction](dmode/driving-mode.md), [Entering Driving Mode](dmode/enter-driving-mode.md), [Features and Operation Logic](dmode/function-and-operation.md), [Working with HiCar](dmode/work-with-hicar.md), [Exiting Driving Mode](dmode/exit-driving-mode.md).

# HiCar Device Description

As the HiCar hardware ecosystem continues to evolve, you can choose the appropriate HiCar device based on your vehicle's specific situation to start using HiCar.

The diagram below simply outlines the general logic for choosing a HiCar device, providing a reference for navigating the complex HiCar device ecosystem.

## 0. How to Check

You can check your vehicle using the methods mentioned in our user manual to determine the most suitable HiCar device. Reference link: [Check Vehicle](/en/docs/check/car).

## 1. Dongle Devices

Using a dongle to access HiCar is likely the simplest and most cost-effective way for existing vehicles — those that do not natively support HiCar — to use the system.

Currently, there are several scenarios for dongle devices, as detailed in the table below.

| No. | Scenario | Link |
| --- | ------------------------------------------- | -------------------------------------- |
| 1   | The vehicle supports Apple's wired CarPlay; a conversion dongle can be used to enable HiCar. | [Vehicle supports wired CarPlay](/en/docs/devices/carplay)     |
| 2   | The vehicle supports Baidu's wired CarLife; a conversion dongle can be used to enable HiCar. | [Vehicle supports wired CarLife](/en/docs/devices/carlife)     |
| 3   | The vehicle runs Android and allows installing apps; a dongle plus an app can be used to enable HiCar. | [Vehicle with Android that allows app installation](/en/docs/devices/android) |

## 2. Aftermarket Simple Screens

If your vehicle does not support connectivity solutions like CarPlay or CarLife, or lacks a screen or has a screen that is too small, you can consider adding an extra screen to use HiCar.

Currently available aftermarket simple screen devices are listed in the table below.

| No. | Scenario | Link |
| --- | -------------------------------------------- | ------------------------------- |
| 1   | If your center console screen is relatively flat, you can consider attaching an in-vehicle smart screen to use HiCar. | [In-Vehicle Smart Screen](/en/docs/devices/smart-screen) |
| 2   | If you are open to a smart rearview mirror form factor, you can use HiCar by adding a smart rearview mirror. | [Smart Rearview Mirror](/en/docs/devices/smart-mirror) |

# Back to Car Interface

After a successful HiCar connection, it provides services in full-screen mode.
When you need to return to the head unit's system interface, first tap the first icon at the top of the Dock area to switch to the app desktop, then tap your car model's icon to return to the head unit system interface.

<img
    src={require('./img/back-to-car-1.png').default}
    width="800" 
/>

If your car model has physical buttons, you can also use them to quickly return to the head unit system interface.

If your car model does not display HiCar in full-screen mode, you can also use other original vehicle-related menus or buttons on the interface to return to the head unit system interface.

# Voice Enhancement

HiCar's Smart Voice has some enhanced capabilities, currently supporting continuous conversation and multi-intent voice commands.

## Continuous Conversation (Barge-in)

Continuous conversation, also known as barge-in or interruption, means that after issuing a command to the HiCar voice assistant, you do not need to wait for the voice assistant to finish speaking before issuing a new voice command — you can interrupt and give the next command immediately.

/>

:::tip
Continuous conversation requires the head unit to have good echo cancellation and noise reduction capabilities. When connecting to HiCar for the first time, there will be a related setup guide interface explaining this. You can check whether the current head unit supports enabling the continuous conversation feature based on the guide.
:::

## Multi-Intent Voice Commands (One Sentence, Multiple Commands)

Multi-intent voice commands, also known as one sentence with multiple commands, means that when issuing instructions to the HiCar voice assistant, you do not need to give one command per sentence. Instead, you can include multiple commands in a single sentence, for example: "Play music and navigate home."

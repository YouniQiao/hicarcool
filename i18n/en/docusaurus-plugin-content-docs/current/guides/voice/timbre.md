# Voice Timbre

HiCar Smart Voice supports syncing the voice timbre settings from the phone's smart voice assistant.

You can set it using any of the following methods when HiCar is not connected:

:::tip

Please set it when HiCar is not connected.

:::

* Wake up the phone's smart voice assistant and speak a command, for example: "Change the voice", "I want to change the voice", etc.
* Go to **Settings > Smart Assistance > Smart Voice > Sound**, swipe left or right through the official sound cards, and tap a card to select your preferred timbre. You can also tap **Add Voice** and follow the on-screen instructions to create a custom voice by recording your own voice or inviting someone else to record.

:::tip

Before using this feature, please go to the AppGallery and update Smart Voice to version 11.0.56.300 or higher.

:::

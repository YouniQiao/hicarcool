# Introduction to Driving Mode

Driving Mode is a new interactive platform designed to provide users with commonly used driving features such as navigation, music, phone, and voice interaction while driving. These functions are provided by their respective apps, but Driving Mode combines them together, similar to how HiCar works in your vehicle.

## Prerequisites

Using HiCar Driving Mode requires meeting the following conditions:
HiCar version 13.2.0.450 or higher.

It does not depend on whether the vehicle's infotainment system supports HiCar.

Interface layout:
* The top section is the navigation window, which can be switched between Amap, Baidu, PetalMaps, etc.
* The middle section contains swipeable cards. Swipe left or right to switch between music, payment, phone, parking assistant, weather cards, etc.
* The bottom section is the Dock area, showing home, map, music, and recently used app icons.

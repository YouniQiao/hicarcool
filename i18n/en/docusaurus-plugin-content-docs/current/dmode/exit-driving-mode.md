# Exiting Driving Mode

There are 3 ways to exit Driving Mode:

First: Tap the first icon in the Dock area to enter the More Apps list, then select the first icon "Exit".

<img
  src={require('./img/exit-driving-mode-1.jpg').default}
  width="400" 
/>

Second: Pull down from the left side of the top navigation bar, and tap "Exit" in the "Driving Mode Active" notification banner.

<img
  src={require('./img/exit-driving-mode-2.jpg').default}
  width="400" 
/>

Third: After returning to the desktop, you will see a HiCar icon attached to the corner of the screen. Drag it to the trash can to exit.

<img
  src={require('./img/exit-driving-mode-3.jpg').default}
  width="400" 
/>

# Enhanced Voice Capabilities

Next HiCar's smart voice assistant inherits the capabilities of Celia on the phone, such as support for multi-intent voice commands.

## Multi-Intent Commands (Multiple Instructions in One Sentence)

Multi-intent voice commands, also known as multiple instructions in one sentence, allow you to issue multiple commands to the HiCar voice assistant in a single sentence instead of one command per sentence. For example: "Play music and navigate home."

# How to Experience HiCar NEXT

For convenience, we will refer to HiCar on the HarmonyOS NEXT (pure-blood Harmony) system as HiCar NEXT.

## What is HiCar NEXT

On October 8, 2024, Huawei began the public beta of HarmonyOS NEXT, allowing a number of Huawei phone models to directly upgrade to the new system.

The new HarmonyOS NEXT system also supports the HiCar function. HiCar on this new system is HiCar NEXT.

HiCar NEXT has some functional differences compared to the HiCar you are currently using. The main reasons are as follows:

* Thanks to the brand-new underlying system, HiCar NEXT can provide some features that were previously difficult to implement with HiCar, such as true app transfer and app split-screen. We believe these unique HiCar NEXT features will continue to grow.
* HarmonyOS NEXT is a new system, and the HiCar NEXT on it has been developed entirely from scratch. Due to time constraints, some features may not be supported temporarily, such as wired connection. However, we believe that over time, HiCar NEXT will gradually catch up in terms of functionality.

For the current feature differences on HiCar NEXT, please refer to [Feature Differences](./diff.md).

* [Feature Differences](./diff.md)

This site will continue to follow the evolution of HiCar NEXT features and provide timely usage instructions.

## Special Notes

HiCar NEXT is primarily a feature upgrade on the phone side. On the vehicle side, whether the original car supports HiCar or you use a connectivity box to access HiCar, it does not affect your experience with HiCar NEXT.

However, it is important to note that if you previously could only use HiCar via a wired connection, since HiCar NEXT currently does not support wired connections, these users will not be able to use HiCar NEXT after upgrading their phone to HarmonyOS NEXT.

:::tip

After upgrading your phone system to HarmonyOS 5, if you need to use HUAWEI HiCar via a wired connection, please upgrade the system to version 5.0.0.126 SP8 or later.

:::

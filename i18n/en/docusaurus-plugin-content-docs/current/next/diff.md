# Feature Differences

Differences in features between HiCar on HarmonyOS NEXT and the previous version on regular HarmonyOS.

Some features are currently missing because they are still under development, so this chapter will be continuously updated as HiCar NEXT's features evolve.

:::tip

Feature difference confirmation date: 2025.8.6

::: 

## Connection

| Feature          | HiCar         | HiCar NEXT                      |
| ----------     | ---------- | ------------------------- | 
| Wireless Connection        | Supported | Supported               | 
| Wired Connection        | Supported | Supported   | 
| Auto Connection        | Supported | Supported | 



## Interface

| Feature          | HiCar         | HiCar NEXT                      |
| ----------- | ---------- | ------------------------- | 
| Desktop Wallpaper        | Supported | Supported               | 
| Wallpaper Desktop       | Not Supported | Supported   | 
| Return to Car Interface        | Supported | Supported | 



## Apps

| Feature          | HiCar         | HiCar NEXT                      |
| ----------- | ---------- | ------------------------- | 
| Adjust App Display Order        | Supported | Supported               | 

## Voice

| Feature          | HiCar         | HiCar NEXT                      |
| ----------- | ---------- | ------------------------- | 
| Voice Function        | Supported | Supported               | 


## Features

| Feature          | HiCar         | HiCar NEXT                      |
| ----------- | ---------- | ------------------------- | 
| Set Home and Work Addresses        | Supported | Supported               | 
| Shake to Transfer Navigation        | Supported | Supported   | 
| Driving Mode        | Supported | Not Supported | 
| Screenshot        | Supported | Not Supported |

# Check Phone

You can check whether your phone supports Next HiCar, or purchase a new phone that supports Next HiCar.

## 1. Which Phone Brands Support Next HiCar

Since Next HiCar is a connectivity solution provided by Huawei, currently only Huawei phones running HarmonyOS 5 and above support Next HiCar.

## 2. Which Phone Models Support Next HiCar

For the specific phone models that support Next HiCar, please check the official website: [Huawei Developer Alliance - HiCar Supported Phones](https://developer.huawei.com/consumer/cn/doc/development/HiCar-Guides/available-phones-0000001227954439)

## 3. Check Whether Your Phone Supports Next HiCar

The best way to check whether your Huawei phone supports Next HiCar is as follows:

Open your phone's "Settings > Multi-Device Collaboration". If you can see the "HUAWEI HiCar" menu, your phone supports Next HiCar.
